Calligrapher

created by @anwarshahriar

A helper library to set custom fonts to your view

Javadoc

This is a simple library for setting specific font to all of your view with a single line. Feel free to fork it or fix any issues and make a pull request.


Integrating this library is really easy you just need to add these two lines of code after calling setContentView()


Above code will set a given custom font to the whole activity including ActionBar. If you want to exclude ActionBar from setting custom font just set false on the setFont method.


If you don't want to set custom font to your whole activity that's okay too. You just need to specific View or ViewGroup to pass into setFont method. Event you can set multiple font to same screen, just remember custom font will be applied to an entire ViewGroup and it's child.
